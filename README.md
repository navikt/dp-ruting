# DP-ruting

Applikasjonen ruter henvendelser fra bruker, avhengig av skjematype og hvor (Arena eller Dagpenger) bruker eventuelt har sak.

## Komme i gang

Gradle brukes som byggverktøy og er bundlet inn.

`./gradlew build`
